# Clonar repositorio en local
Pasos por si alguno tiene problemas para iniciar el proyecto frontend clonado del repositorio.

Tener node.js 14 y angular12 instalado

1. En crhome iniciar sesión en github
2. Iniciar vcode
3. En vcode en el icono account (inferior derecha) iniciar con git
4. Autorizar y reiniciar vcode
5. Clic en view -> command palette - pegar la url  del repositorio
6. Seleccionar el directorio donde se va a trabajar
7. en vcode ctrl + ñ iniciar terminal (ver que es la ruta correcta)
8. npm install --save @angular-devkit/build-angular (esperar que instale módulos)
9. ng serve
